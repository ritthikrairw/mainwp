=== MainWP WooCommerce Status Extension ===
Plugin Name: MainWP WooCommerce Status Extension
Plugin URI: https://mainwp.com
Description: MainWP WooCommerce Status provides you a quick overview of your WooCommerce stores in your network. Requires MainWP Dashboard plugin.
Version: 4.0.8
Author: MainWP
Author URI: https://mainwp.com

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP WooCommerce Status Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-woocommerce-status-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP WooCommerce Status Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
