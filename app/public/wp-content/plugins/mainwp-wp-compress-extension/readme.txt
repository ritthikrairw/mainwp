== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP WP Compress Extension (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-wp-compress-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP WP Compress Extension plugin through the 'Plugins' menu in WordPress
