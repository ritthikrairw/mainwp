=== MainWP WooCommerce Shortcuts Extension ===
Plugin Name: MainWP WooCommerce Shortcuts Extension
Plugin URI: https://mainwp.com
Description: MainWP WooCommerce Shortcuts provides you a quick access WooCommerce pages on your child sites.
Version: 4.1.2
Author: MainWP
Author URI: https://mainwp.com

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP WooCommerce Shortcuts Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-woocommerce-shortcuts-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP WooCommerce Shortcuts Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
