=== MainWP Page Speed Extension ===
Plugin Name: MainWP Page Speed Extension
Plugin URI: https://mainwp.com
Description: MainWP Page Speed Extension enables you to use Google Pagespeed Insights to monitor website performance across your WordPress websites.
Version: 4.0.2
Author: MainWP
Author URI: https://mainwp.com
Icon URI:

== Installation ==
1. Please install plugin the MainWP Dasbhoard and active it before install MainWP Page Speed Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-page-speed-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP Page Speed Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
