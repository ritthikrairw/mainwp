=== Clone Extension ===
Plugin Name: MainWP Clone Extension
Plugin URI: https://mainwp.com
Description: MainWP Clone Extension is an extension for the MainWP plugin that enables you to clone your child sites with no technical knowledge required.
Version: 4.0.2
Author: MainWP
Author URI: https://mainwp.com

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install Clone Extension plugin (get the MainWP Dashboard plugin from url:http://mainwp.com/)
1. Upload the `mainwp-clone-extension` folder to the `/wp-content/plugins/` directory
1. Activate the Clone Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
