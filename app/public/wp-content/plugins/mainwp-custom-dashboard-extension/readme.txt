== Description ==

The MainWP Custom Dashboard Extension is designed to contain your custom snippets for your [MainWP Dashboard](https://wordpress.org/plugins/mainwp/) plugin. By adding custom snippets to this plugin, you can easily extend your MainWP Dashboard functionality or change the way it looks.

It will help you to keep all your changes in one location and independent of the other components. This will enable you to safely update your MainWP Dashboard plugin on your site without worry of losing your modifications as well as deactivating your custom code if needed for troubleshooting.

For the MainWP function reference, please check the [MainWP Codex](https://codex.mainwp.com).

== Installation ==

1. Upload `mainwp-custom-dashboard-extension` to the `/wp-content/plugins/` directory on your MainWP Dashboard site
2. Add your custom snippets to the plugin either locally or via Plugins > Editor in your MainWP Dashboard site
3. Activate the plugin through the 'Plugins' menu in WordPress
