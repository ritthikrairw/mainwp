== Installation ==
1. Please install plugin the MainWP Dasbhoard and active it before install MainWP Domain Monitor Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-domain-monitor-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP Domain Monitor Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
