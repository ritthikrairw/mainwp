Author URI: https://mainwp.com
Documentation URI: https://mainwp.com/help/docs/category/mainwp-extensions/virusdie/

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP Virusdie Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com)
2. Upload the `mainwp-virusdie-extension` folder to the `/wp-content/plugins/` directory
3. Activate the Virusdie Extension plugin through the 'Plugins' menu in WordPress

1. Enable or Disable extension on the "Extensions" page in the dashboard
