=== Custom Post Types Extension ===
Plugin Name: Custom Post Types Extension
Plugin URI: https://mainwp.com
Description: MainWP Extension for managing Custom Post Types (CPTs)
Version: 4.0.3
Author: MainWP
Author URI: https://mainwp.com

== Description ==

Custom Post Types Extension is an extension for the MainWP Plugin that allows you to manage almost any custom post type on your child sites which includes Publishing, Editing and Deleting custom post type content.

== Installation ==

1. Please install plugin "MainWP Dashboard" and active it before install Custom Post Types Extension plugin ( get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-custom-post-types` folder to the `/wp-content/plugins/` directory
3. Activate the Extension plugin through the 'Plugins' menu in WordPress
