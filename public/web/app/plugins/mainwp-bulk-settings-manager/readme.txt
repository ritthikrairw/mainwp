=== MainWP Bulk Settings Manager Extension ===
Plugin Name: MainWP Bulk Settings Manager Extension
Plugin URI: https://mainwp.com
Description: The Bulk Settings Manager Extension unlocks the world of WordPress directly from your MainWP Dashboard. With Bulk Settings Manager you can adjust your Child site settings for the WordPress Core and almost any WordPress Plugin or Theme.
Version: 4.0.4
Author: MainWP
Author URI: https://mainwp.com

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP Bulk Settings Manager Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-bulk-settings-manager` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP Bulk Settings Manager Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
