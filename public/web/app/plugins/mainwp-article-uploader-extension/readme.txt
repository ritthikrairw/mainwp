=== MainWP Article Uploader Extension ===
Plugin Name: MainWP Article Uploader Extension
Plugin URI: https://mainwp.com
Description: MainWP Article Uploader Extension allows you to bulk upload articles to your dashboard and publish to child sites.
Version: 4.0.1.1
Author: MainWP
Author URI: https://mainwp.com

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP Article Uploader Extension plugin (get the MainWP Dashboard plugin from url:http://mainwp.com/)
2. Upload the `mainwp-article-uploader-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP Article Uploader Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
