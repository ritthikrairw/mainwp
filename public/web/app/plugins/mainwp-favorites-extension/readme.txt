=== MainWP Favorites Extension ===
Plugin Name: MainWP Favorites Extension
Plugin URI: https://mainwp.com
Description: MainWP Favorites is an extension for MainWP main plugin that allow you manage yours favorites plugins and themes on child sites.
Version: 4.0.10
Author: MainWP Plugins
Author URI: https://mainwp.com

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install Favorites Extension plugin( get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-favorites-extension` folder to the `/wp-content/plugins/` directory
3. Activate the Favorites Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on Manage Extension Panel
2. Set "Favorites Themes and Plugins" in Extensions Menu with following functions:
     2.1. Manage group of favorites plugins/themes
     2.2. Install favorites plugins/themes
3. Add plugins/theme into your favorites at plugin screen or theme screen.
