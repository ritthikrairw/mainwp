=== MainWP File Uploader Extension ===
Plugin Name: MainWP File Uploader Extension
Plugin URI: https://mainwp.com
Description: MainWP File Uploader Extension gives you an simple way to upload files to your child sites.
Version: 4.1
Author: MainWP
Author URI: https://mainwp.com

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install Uploader Extension plugin (get the MainWP Dashboard from url:https://mainwp.com/)
2. Upload the `mainwp-file-uploader-extension` folder to the `/wp-content/plugins/` directory
3. Activate the File Uploader Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
