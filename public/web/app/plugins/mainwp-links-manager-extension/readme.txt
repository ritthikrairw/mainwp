Plugin Name: MainWP Links Manager Extension
Plugin URI: https://mainwp.com
Description: MainWP Links Manager is an Extension that allows you to create, manage and track links in your posts and pages for all your sites right from your MainWP Dashboard.
Version: 2.0
Author: MainWP
Author URI: https://mainwp.com
Documentation URI: https://mainwp.com/help/category/mainwp-extensions/links-manager/

== Installation ==
1. Please install plugin "MainWP Dashboard" and active it before install MainWP Links Manager Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-links-manager-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP Links Manager Extension plugin through the 'Plugins' menu in WordPress
